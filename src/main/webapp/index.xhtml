Movie and lists can be viewed by simple offset pagination and field search capabilities. Movie lists are sorted by movie title while actors by last and first name. Beside browsing editing is also supported for all three entity types. Integrated CDI Events and WebSockets provide highly interactive picture about persistence layer to browser clients. In contrast to this JSF interface application also provides basic entity operations over a REST interface.

Actor-List

Lists actors with paging and search capabilities. Provides removing of actors or selecting them as cast candidates. Facilitates individual actor selection with navigation to Actor-Details view.

Movie-List

Lists actors with paging and search capabilities. Provides removing of movies or selecting them as cast candidates. Facilitates individual movie selection with navigation to Movie-Details view.

Actor-Details

Displays individual actor with a list of casted movies. Provides basic create, update and delete operations to actors. Provides casting operations towards movies.

Movie-Details

Displays individual movie with a list of casted actors and associated images. Provides basic movie create, update and delete operations to actors and removal of images. Provides casting operations towards actors. Facilitates individual image selection with navigation to Image view.

Image

Displays image details; attributes including image itself. Provides basic create with upload, update and delete operations.
